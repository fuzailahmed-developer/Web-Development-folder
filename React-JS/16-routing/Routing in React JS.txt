<Route /> best practice self closing
	* ka kaam kya hai?
	* <Route> ek individual path aur uske component ko define karta hai.
	* Har <Route> ke 2 main props hote hain:
	# path → kis URL pe ye route trigger hoga
	# element → konsa component show karna hai
	• <Route path='/' element={<Home />}/>
