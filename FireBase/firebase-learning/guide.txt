2. 2 types of Database realtime and firestore 

| Feature         | Firestore               | Realtime Database   |
| --------------- | ----------------------- | ------------------- |
| Data Model      | Documents & Collections | Large JSON tree     |
| Scalability     | ⭐⭐⭐⭐⭐            | ⭐⭐⭐             |
| Real-time Speed | ⭐⭐⭐⭐               | ⭐⭐⭐⭐⭐ (fastest)|
| Query Power     | ⭐⭐⭐⭐⭐             | ⭐⭐                |
| Security Rules  | Easy                    | Hard                |
| Price           | Reads/Writes            | Bandwidth           |
| Best For        | Large apps, dashboards  | Chat, live tracking |

firestore 
import { getFirestore } from "firebase/firestore";
export const db = getFirestore(app)

realtime
import { getDatabase } from "firebase/database";
